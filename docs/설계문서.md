# 피클볼 회원관리 시스템 — 설계 문서

## 개요
PC(웹 관리자)와 모바일 앱(PWA)이 하나의 Supabase DB를 공유. GitHub Pages로 무료 배포.

## 기능
- 회원가입 신청 → 관리자 승인 (이름/전화/생년월일/추천인)
- 스케줄 달력 + 날짜별 시간표(1시간 단위) 예약 신청 → 관리자 승인
- 회비 달력: 날짜별 입금인 / 미입금인(입금 예정), 1~12개월 납부로 다음 예정일 자동 계산
- 코칭 스탭 연락처·주소 관리
- 실시간 동기화 + 관리자 알림(벨 배지/토스트/데스크톱 알림)
- 카카오 알림톡 발송(선택)

## 기술 스택
- 프런트엔드: 순수 HTML/CSS/JS (빌드 불필요), PWA
- 백엔드: Supabase (PostgreSQL + Auth + Realtime)
- 알림: Supabase Edge Function → 카카오 알림톡 대행사

## 데이터 모델
coaches, members, schedules, bookings, payments, notifications. 상세는 supabase/schema.sql 참고.

## 로드맵
1단계(완료): DB, 동작 웹앱, 실시간/알림, 배포 준비.
2단계: 로그인(회원/관리자 구분) + RLS 강화.
3단계: 회비 만료 자동 알림(크론).
4단계: 카카오 알림톡 실제 연결.
5단계: 통계, 회원 셀프 예약, 결제 연동.
