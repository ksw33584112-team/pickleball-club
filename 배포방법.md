# 🚀 배포 방법

3단계: ① Supabase 테이블 만들기 → ② 연결 키 넣기 → ③ GitHub Pages 공개.

## 1단계 · Supabase
1. supabase.com 로그인 → 프로젝트 선택(없으면 New project, Region: Seoul)
2. SQL Editor → New query → `supabase/schema.sql` 전체 붙여넣고 **Run**

## 2단계 · 연결 키
1. Project Settings → API 에서 **Project URL**, **anon public** 키 복사
2. `assets/js/config.js` 의 두 줄에 붙여넣기:
   ```js
   SUPABASE_URL: "https://xxxx.supabase.co",
   SUPABASE_ANON_KEY: "eyJ...",
   ```
3. `index.html` 새로고침 → 우측 상단 "Supabase 연결됨" 확인

## 3단계 · GitHub Pages
1. github.com → New repository (예: pickleball-club, Public)
2. 저장소에 이 폴더 안의 파일들을 업로드(드래그) → Commit
3. Settings → Pages → Deploy from a branch → main / (root) → Save
4. 1~2분 뒤 나오는 `https://(아이디).github.io/(저장소)/` 가 완성 주소

## 4단계(선택) · 카카오 알림톡
사업자 등록 + 카카오 채널 + 발송 대행사(솔라피 등) + 템플릿 승인 후:
```
supabase functions deploy send-alimtalk
supabase secrets set SOLAPI_API_KEY=... SOLAPI_API_SECRET=... KAKAO_PF_ID=... KAKAO_TEMPLATE_ID=... SENDER_PHONE=...
```

## 배경 사진 바꾸기
`assets/images/hero.jpg` 를 원하는 사진(같은 이름)으로 덮어쓰면 메인화면 배경이 바뀝니다.

## 로그인(다음 단계)
지금은 누구나 접근 가능(anon 허용). 회원/관리자 로그인을 붙인 뒤 schema.sql의 anon 정책 블록을 제거하면 보안이 강화됩니다.
